Buscamos colaboradores para Vasak OS: ayudanos a construir una nueva experiencia Linux

Vasak OS es un proyecto open source que busca construir una distribución Linux moderna basada en Arch Linux, enfocada en ofrecer una experiencia de usuario completa, coherente y bien integrada.

El objetivo del proyecto es desarrollar un ecosistema de aplicaciones y herramientas que permitan usar el sistema de forma cómoda y moderna sin depender constantemente de la terminal, manteniendo al mismo tiempo la potencia y flexibilidad del ecosistema Linux.

Actualmente el proyecto se encuentra en una etapa temprana de desarrollo (pre-alpha), pero ya cuenta con una ISO funcional y varios componentes del sistema en desarrollo.

Para poder seguir avanzando necesitamos sumar personas interesadas en colaborar.

Este es un proyecto sin fines de lucro, impulsado por la comunidad y abierto a cualquier persona que quiera aportar.

¿Qué es Vasak OS?

Vasak OS es una distribución basada en Arch Linux que busca ofrecer:

  • Un desktop environment propio
  • Aplicaciones diseñadas específicamente para el sistema
  • Una experiencia visual consistente
  • Integración entre aplicaciones
  • Herramientas que simplifiquen el uso del sistema

Actualmente se están desarrollando varios componentes del sistema, incluyendo:

  • Desktop Environment propio
  • Panel del sistema
  • Menú de aplicaciones
  • Centro de notificaciones
  • Applets
  • File Manager
  • Aplicaciones adicionales que irán apareciendo con el tiempo

La distribución utiliza Arch Linux como base, lo que permite contar con un sistema moderno y actualizado.

El estado actual del proyecto

El proyecto se encuentra en una etapa pre-alpha. Esto significa que todavía falta mucho trabajo por hacer, pero ya existe una base funcional:

  • ISO del sistema
  • Desktop environment en desarrollo
  • File manager propio
  • Infraestructura inicial del proyecto

La ISO se genera utilizando archiso, y el sistema cuenta con repositorios propios de paquetes.

Queremos sumar colaboradores

Actualmente el proyecto está siendo desarrollado por una sola persona, por lo que sumar colaboradores puede marcar una gran diferencia en la evolución del sistema.

No buscamos únicamente expertos.

Si te interesa el software libre y querés aprender o aportar, tu ayuda es bienvenida.

Perfiles que estamos buscando

Desarrolladores

El desarrollo de las aplicaciones del sistema se realiza principalmente con:

  • Tauri
  • VueJS
  • TypeScript

Buscamos desarrolladores interesados en colaborar en:

  • Aplicaciones del sistema
  • Herramientas del desktop
  • Nuevas apps para el ecosistema Vasak

No es necesario ser experto, pero sí tener interés en aprender y contribuir.

Diseñadores UI / UX

La experiencia de usuario es uno de los pilares del proyecto. Buscamos personas interesadas en trabajar en:

  • Diseño de interfaces
  • Experiencia de usuario
  • Consistencia visual del sistema
  • Mejoras en la interacción entre aplicaciones

Diseñadores gráficos

Necesitamos ayuda para mejorar la identidad visual del proyecto. Algunas tareas incluyen:

  • Diseño de icon packs
  • Material visual para redes sociales
  • Mejoras visuales del sitio web
  • Recursos gráficos del sistema

Content Managers

El proyecto también necesita mantener una presencia activa en su sitio web. Buscamos personas que puedan colaborar en:

  • Redacción de artículos
  • Actualizaciones del sitio
  • Documentación del proyecto
  • Publicación de novedades

El sitio está desarrollado con Hugo CMS.

Community Managers

La comunidad es clave para el crecimiento del proyecto. Buscamos personas interesadas en:

  • Moderar y dinamizar la comunidad
  • Mantener actividad en redes
  • Conectar las necesidades de los usuarios con el desarrollo del proyecto

Actualmente estamos presentes en:

  • Telegram
  • Discord
  • Reddit

Testers

Los testers ayudan a detectar errores y mejorar la estabilidad del sistema.

Si te interesa probar software en desarrollo y reportar problemas, esta es una forma excelente de contribuir.

Maintainers de paquetes

La distribución cuenta con repositorios propios de paquetes. Buscamos personas que puedan colaborar en:

  • Mantenimiento de paquetes
  • Actualización de repositorios
  • Mantenimiento de las ISOs cuando requieran actualización

Principalmente trabajamos con paquetes para pacman, aunque también existen repositorios APT.

DevOps / Infraestructura

El proyecto también necesita personas interesadas en:

  • Infraestructura del proyecto
  • Automatización de builds
  • CI/CD
  • Servidores y repositorios de paquetes

Documentación

Una buena documentación es fundamental para cualquier proyecto open source. Buscamos colaboradores que puedan ayudar con:

  • Guías de instalación
  • Documentación del sistema
  • Tutoriales
  • Wiki del proyecto

QA / Testing automatizado

También sería valioso contar con personas que puedan colaborar en:

  • Automatización de tests
  • Validación de builds
  • Mejora de la calidad del software

Seguridad

Personas interesadas en colaborar en:

  • Revisión de dependencias
  • Prácticas de seguridad
  • Mejoras de hardening del sistema

Traducciones y localización

En el futuro queremos que el proyecto esté disponible en múltiples idiomas. Si te interesa colaborar con traducciones, tu ayuda también será bienvenida.
