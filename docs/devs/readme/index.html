Stack Tecnológico

  • Frontend: Vue.js 3.5, TypeScript, Tailwind CSS
  • Backend: Rust 1.70+, Tauri 2.8+
  • Comunicación: D-Bus, IPC de Tauri
  • Build: Vite, Cargo
  • Package Manager: Bun (frontend), Cargo (backend)
