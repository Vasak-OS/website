Bienvenidos a Vasak OS

Un sistema basado en GNU/Linux y pensado para vos

Linux

Utilizamos todo el potencial y la seguridad de Linux para ofrecerte una experiencia única.

Open Source

Todo nuestro código es abierto y puedes verlo en GitHub. ¡Incluso puedes contribuir!

Framework VueJS

Nuestra Interfaz de Usuario está desarrollada con VueJS, un framework moderno y potente.

Responsive

Nuestro sistema se adapta a cualquier dispositivo, ya sea un móvil, una tablet o un ordenador.

Personalizable

Al usar CSS para el estilo, el sistema es altamente personalizable y puedes cambiarlo a tu gusto.

Python

Python es un lenguaje relativamente sencillo y potente, por lo que es ideal para el desarrollo de aplicaciones.

Forma parte de esta gran comunidad

Podes formar parte de esta gran comunidad descargando nuestro sistema y utilizándolo, probándolo, o incluso unirte al desarrollo del mismo. Vasak OS lo hacemos entre todos, desde los usuarios hasta los desarrolladores.

Todo nuestro trabajo en Vasak es completamente abierto y libre para toda la comunidad. Cualquier usuario puede intervenir con el proyecto desde muchas posiciones y llevar el sistema hacia su propio ideal.

Ya seas diseñador, programador, usuario tu ayuda es más que bienvenida y siempre se puede sumar a un proyecto. Recuerda que desde que instalas Vasak OS ya sos parte vital de nuestra comunidad.

Vasak OS a diferencia de otras “distros” de GNU/Linux no busca reducir los recursos. En consecuencia busca realizar una buena precarga y un óptimo uso de los recursos para mejorar la experiencia de uso y la fluidez del sistema.

Por la propia naturaleza de Arch Linux y apoyándonos en ellos podemos ofrecer un sistema completamente actualizado que no necesita fragmentarse en versiones. De esta forma todos los usuarios siempre podrán acceder al software más nuevo.

  • Sistema flexible.
  • Software Actualizado.
  • Parches de seguridad continuos.
