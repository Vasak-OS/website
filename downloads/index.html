Verificacion de integridad

Ejecuta uno de estos comandos en el directorio donde descargaste la ISO y compara el resultado con el SHA256 publicado.

Linux

sha256sum vasakos-2026.04.15-x86_64.iso

macOS

shasum -a 256 vasakos-2026.04.15-x86_64.iso

Windows

CertUtil -hashfile vasakos-2026.04.15-x86_64.iso SHA256
