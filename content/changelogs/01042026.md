* Se reduce el consumo del desktop
* Se mejora proceso de compilacion
* Se actualizan dependencias

* Se mejora la integracion con el desktop

## `vasak-file-manager` [BETA]

* Se encuentra en disponible la nueva [App de File Manager](/blog/2026/03/vasak-file-manager-un-nuevo-explorador-de-archivos-para-vasak-os/)
* Mejoras en la navegacion y rendimiento
* Mejoras en la integracion con el desktop
* Sistemas de pestañas
* Multiples Vistas

## ISO base arch [`archiso`]

* Se agrega Calamares para poder instalar el sistema operativo de forma persistente
* Se eliminan dependencias antiguas
* Se mejora el proceso de build de la ISO

## Calamares

* Se agrega modulo de Bootloader para poder instalar GRUB y configurar el arranque del sistema operativo
* Se configura compresiones de discos para optimizar el espacio en disco
* Se mejora la experiencia de usuario en el proceso de instalacion

> ATENCION: De momento `calamares` no funciona desde el menu y es necesario ejecutarlo desde la terminal con el comando `sudo calamares` para poder iniciar el proceso de instalacion. Esto se debe a que estamos en una etapa temprana de desarrollo y estamos trabajando para mejorar la integracion del instalador con el sistema operativo. Cualquier reporte o colaboracion para mejorar esta experiencia es mas que bienvenida.
